14-Glassy

14-Glassy is derived from the WordPress 2014 official theme, fine-tuned for mobile devices to improve readability. Support native color scheme and Fourteen Colors Plugin. See website for demo.

Feature

  1. Transparency for entry headers, which make presentation smoother.
  2. Support both native color scheme and Fourteen Colors Plugin.
  3. Visible areas are fine-tuned for best readability.
  4. Trivial elements are hidden when using small screen.
